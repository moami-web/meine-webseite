Microsoft 365 Prozessautomatisierung Webseite
Statische persönliche Dienstleister-Webseite für SharePoint, Power Automate, PowerApps, Microsoft 365, einfache interne Web-Tools und alltägliche IT-Unterstützung.

Lokal öffnen
Repository herunterladen oder klonen.
index.html direkt im Browser öffnen.
Die Seite verwendet nur HTML, CSS und JavaScript. Es werden keine externen Bibliotheken, Frameworks oder Bilder geladen.

Dateien
index.html - semantische Seitenstruktur und Inhalte
style.css - responsives Layout, Farben, Animationen und Komponenten
script.js - mobiles Menü, aktiver Navigationspunkt und Scroll-Reveal
GitHub Pages veröffentlichen
In GitHub das Repository öffnen.
Settings auswählen.
Unter Pages als Source Deploy from a branch wählen.
Branch main und Ordner /root auswählen.
Speichern und warten, bis GitHub Pages die Seite veröffentlicht.
Wichtig: index.html liegt im Hauptverzeichnis, damit GitHub Pages die Seite direkt ausliefern kann.
